<h1 align="center">PakitoBomberBOT</h1>

PakitoBomberBOT es un **CallBomber** exclusivo para España que permite realizar llamadas masivas a un número de teléfono de forma automatizada.


### Funcionalidades:
- [x] **Modo Automático**: Ejecución continua de solicitudes de llamada.
- [x] **Exclusivo España**: Optimizado para servicios y prefijos españoles.

### Dependencias:
- Firefox o Chrome
- selenium
- requests
- colorama
- console_menu

### Instalación:
```bash
pip install -r requirements.txt
```

### Ejecución:
```bash
python PakitoBomberBOT.py
```
